#Windows使用ssh远程连接虚拟机服务器

ssh

SSH 通过在网络中创建安全隧道来实现 SSH 客户端与服务器之间的连接，SSH是一种加密传输协议，以非对称加密实现身份验证，最常见的用途是远程登录系统。

步骤

1. 配置网络

  • 本地和虚拟机须在同一个网段下，保证主机和虚拟机能够互相ping通，
    在此使用的网络连接模式为NAT模式.

    2. 安装ssh

  • 检测是否安装ssh服务

ssh localhost

提示ssh:connect to host localhost port 22: Connection refused，则表示未安装

  • 安装ssh服务

sudo apt-get install openssh-server

  • 单次开启ssh

sudo systemctl start ssh

  • 开机自启ssh

sudo systemctl enable ssh

  • 单次关闭ssh

sudo systemctl stop ssh

  • 关闭ssh自启命令

sudo systemctl disable ssh

  • 查看ssh服务是否开启

sudo ps -e|grep ssh 有sshd，说明ssh服务已开启，ssh-agent表示ssh-client启动，sshd表示ssh-server启动了.

  1. 用ssh客户端工具进行连接，成功远程控制虚拟机中的服务器

与此无关的一个小tips

好久没用ubuntu，在切换到根用户时，总是密码错误，还以为是因为自己忘记了root密码，后来搜了下才知道，是自己没有初始root密码，Ubuntu的默认root密码是随机的，即每次开机都有一个新的root密码.

解决

输入sudo passwd root，重置密码就可以了.
